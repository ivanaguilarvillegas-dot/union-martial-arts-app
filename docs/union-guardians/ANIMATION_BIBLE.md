# UNION GUARDIANS - ANIMATION BIBLE
Version: 0.1 (Scaffold)
Status: Active - Structure only
Scope: Character and gameplay motion standards

## Purpose
Define animation philosophy, readability, personality expression, and production constraints for all animated assets.

## What This Document Will Contain
- Motion identity and timing doctrine.
- Character acting rules by archetype.
- Gameplay readability thresholds.
- Combat animation principles.
- Loop quality standards.
- Integration with VFX and audio cues.

## What Is Not Included Yet
- Final animation set lists.
- Final timing sheets.
- Final rig-specific constraints.
- Final state machine mapping.

## Development Stages
1. Stage A: Motion pillars and readability.
2. Stage B: Character acting and combat standards.
3. Stage C: Technical constraints and integration.
4. Stage D: Polish and consistency review.
