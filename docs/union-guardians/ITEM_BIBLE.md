# UNION GUARDIANS - ITEM BIBLE
Version: 0.1 (Scaffold)
Status: Active - Structure only
Scope: Item ecosystem and economy-facing design

## Purpose
Define item taxonomy, rarity logic, utility, acquisition channels, and narrative alignment.

## What This Document Will Contain
- Item categories and role definitions.
- Economy and progression alignment.
- Crafting and upgrade pathways.
- Drop logic and fairness model.
- Visual and naming standards.
- Anti-pay-to-win guardrails.

## What Is Not Included Yet
- Final item lists.
- Final drop rates.
- Final crafting recipes.
- Final live-ops event item matrix.

1. Stage A: Item philosophy and categories.
2. Stage B: Acquisition and economy rules.
3. Stage C: Crafting, upgrades, and rarity logic.
4. Stage D: Balance and exploitation safeguards.
