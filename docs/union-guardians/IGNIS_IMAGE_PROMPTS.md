## 1) Prompt - 12 conceptos iniciales
```text
Create a professional AAA concept sheet with 12 clearly different early design explorations of IGNIS, an original fire guardian character for the Union Guardians universe. Show 12 full-body thumbnails on one page, each with unique silhouette direction but all preserving core identity: biped stylized reptilian guardian, upward-triangle silhouette, broad forearms, grounded legs, segmented flame tail, ceremonial combat armor, chest ember core. Keep thermal crest plates on head (not horns). Visual tone: heroic, readable for kids and epic for teens. Include concept labels A01 to A12. Focus on shape language exploration, armor variation, and personality through posture.

Art direction: AAA stylized fantasy, clean production linework, strong silhouette readability, premium concept art quality.

Hard constraints: no franchise imitation, no dragon horns, no wings, no military heavy armor, no photoreal gore, no comedic chibi proportions.

Background: neutral light gray studio concept board.
```

## 2) Prompt - 6 versiones refinadas
```text
Using the best directions from previous IGNIS exploration, generate 6 refined concept variants on a single polished concept sheet. Each variant must feel production-ready and distinct in armor rhythm, crest shape details, chest core architecture, and tail flame behavior, while preserving the official IGNIS identity.

Show each variant in 3/4 front hero view, with small callouts for: silhouette logic, armor tier, and fire motif.

Style: AAA stylized concept art, high readability, clean rendering, premium game character development quality.

Keep core canon: disciplined courage, protector-striker role, ceremonial armor, thermal crest plates (not horns), controlled fire energy.

Avoid: copycat designs, overcomplicated clutter, generic monster look, toy-like plastic surfaces.
```

## 3) Prompt - diseño final AAA
```text
Create the final definitive AAA character concept of IGNIS for Union Guardians. One hero full-body render in 3/4 front view, production quality, cinematic but readable. IGNIS must feel like a flagship franchise character: disciplined fire guardian, noble protector, battle-ready.

Required features: rounded-triangle head with thermal crest plates, almond eyes with ember glow, chest ember core emblem, compact ceremonial armor with layered shoulder and forearm guards, stable combat legs, segmented flame tail, minimal cloth accents, fire effects controlled and directional.

Materials: semi-matte volcanic skin, brushed heat-forged alloy, pyro-crystal chest core.
Lighting: warm key + controlled rim light. Keep form readable.

Output quality: top-tier AAA stylized game concept render, original IP identity, no imitation.
```
